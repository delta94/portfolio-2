Facebook Clone
Live Api code Frontend code

Social media app where users can create an account and add friends by sending them requests which the receiver must accept to become friends.

They can create posts to share with friends through the live feed, posts can have reactions and comments. Users can edit or delete their own posts. Each user has a profile page where they can see all their posts and edit their profile information.

Friends can chat with each other through private messaging.
